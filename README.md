## üîµ Phase 1: The Trinity (Fondasi Web) (Bulan 1-2)
*Fokus: HTML5, CSS3, & JavaScript Dasar. Jangan loncat ke Framework sebelum fase ini!*

### HTML5 (Struktur)
- [ ] ‚úÖ **Semantic HTML:** `<header>`, `<main>`, `<article>`, `<footer>` (Penting untuk SEO & Aksesibilitas).
- [ ] ‚úÖ **Forms & Validation:** Input types, attributes, basic validation.
- [ ] ‚úÖ **SEO Basic:** Meta tags, Open Graph (Agar link bagus saat dishare di WA/FB).
- [ ] ‚≠ê **Accessibility (A11y):** ARIA labels (Nilai plus untuk klien korporat).

### CSS3 (Tampilan)
- [ ] ‚úÖ **Box Model:** Margin, Border, Padding, Content (Wajib paham ini sampai mimpi!).
- [ ] ‚úÖ **Layouting:** Flexbox (Wajib), Grid (Dasar).
- [ ] ‚úÖ **Responsive Design:** Media Queries (`@media`), Mobile-first approach.
- [ ] ‚úÖ **Units:** `rem`, `em`, `vh`, `vw`, `%` (Jangan kebanyakan pakai `px`).
- [ ] ‚≠ê **Animation:** Transitions, Keyframes (Untuk efek "wow" sederhana).
- [ ] ‚ö†Ô∏è **Jangan:** Menghabiskan waktu untuk membuat desain dari nol di Figma jika Anda bukan desainer. Gunakan referensi.

### JavaScript (Logika & Interaksi)
- [ ] ‚úÖ **DOM Manipulation:** `querySelector`, `addEventListener`, `classList`.
- [ ] ‚úÖ **ES6+ Basics:** `let/const`, Arrow Functions, Template Literals.
- [ ] ‚úÖ **Fetch API:** Mengambil data dari API publik (Contoh: menampilkan data cuaca/kurs).
- [ ] ‚≠ê **Local Storage:** Menyimpan data sederhana di browser.
- [ ] ‚ö†Ô∏è **Batas Waktu:** Maksimal 2 bulan. Setelah bisa bikin website statis responsif + interaksi sederhana, **STOP** dan lanjut ke Framework.
